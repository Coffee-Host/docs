# 超大咖託管文檔

這是一個使用 [Create Fumadocs](https://github.com/fuma-nama/fumadocs) 生成的 Next.js 應用程式。

安裝套件並執行開發伺服器：

```bash
npm install
npm run dev
```

在瀏覽器中開啟 http://localhost:3000 查看結果。
