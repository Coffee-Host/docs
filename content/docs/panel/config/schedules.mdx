    ![圖片](/images/panel/config/schedules/create.png)

    ![圖片](/images/panel/config/schedules/create2.png)
    </Step>
    <Step>
    點擊「管理排程」
    ![圖片](/images/panel/config/schedules/manage.png)
    </Step>
    <Step>
    點擊「新工作」
    ![圖片](/images/panel/config/schedules/new.png)
    </Step>
    <Step>
    設定完成後點擊「創建任務」
    ![圖片](/images/panel/config/schedules/create-task.png)
    </Step>
</Steps>
## 範例

### 每日午夜自動重啟

    - 排程設定為 0 0 * * *
    - 任務動作選擇「傳送電源操作」
    - 負載選擇「重啟伺服器」
