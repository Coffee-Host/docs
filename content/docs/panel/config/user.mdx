<Callout title="什麼是子用戶？">
子用戶功能讓您可以邀請其他人共同管理伺服器，並且可以精細控制每位子用戶的權限。
</Callout>

# 子用戶功能介紹

子用戶可以用於以下情境：
- 邀請朋友幫忙管理伺服器
- 讓團隊成員協作管理
- 授權特定人員執行特定操作（如備份、重啟等）

# 新增子用戶

<Steps>
    <Step>

    ### 進入使用者管理頁面

    1. 前往面板的「使用者」頁面
    2. 您會看到目前所有子用戶的列表

    ![使用者管理頁面](/images/panel/config/user/list.png)

    </Step>

    <Step>

    ### 點擊「新增使用者」

    點擊頁面上的「新增使用者」按鈕，開啟新增子用戶的視窗。

    ![新增使用者按鈕](/images/panel/config/user/add-button.png)

    </Step>

    <Step>

    ### 輸入子用戶的電子郵件

    輸入您想邀請的用戶的電子郵件地址。

    ![輸入電子郵件](/images/panel/config/user/email.png)

    </Step>

    <Step>

    ### 設定子用戶權限

    根據您的需求勾選要授予的權限。權限分為多個類別：

    ![設定權限](/images/panel/config/user/permissions.png)

    </Step>

    <Step>

    ### 確認新增

    檢查設定無誤後，點擊「邀請使用者」完成新增。子用戶將會收到邀請通知。

    </Step>

</Steps>

# 權限說明

以下是各項權限的詳細說明：

## 控制權限

| 權限名稱 | 說明 |
|----------|------|
| 控制台 | 查看控制台輸出並執行指令 |
| 啟動 | 啟動伺服器 |
| 停止 | 停止伺服器 |
| 重新啟動 | 重新啟動伺服器 |

## 使用者權限

| 權限名稱 | 說明 |
|----------|------|
| 建立 | 建立新的子用戶 |
| 讀取 | 查看子用戶列表 |
| 更新 | 修改子用戶權限 |
| 刪除 | 移除子用戶 |

## 檔案權限

| 權限名稱 | 說明 |
|----------|------|
| 建立 | 建立新檔案或資料夾 |
| 讀取 | 查看和下載檔案 |
| 更新 | 編輯檔案內容 |
| 刪除 | 刪除檔案或資料夾 |
| 封存 | 壓縮和解壓縮檔案 |
| SFTP | 使用 SFTP 連線存取檔案 |

## 備份權限

| 權限名稱 | 說明 |
|----------|------|
| 建立 | 建立新備份 |
| 讀取 | 查看備份列表 |
| 刪除 | 刪除備份 |
| 下載 | 下載備份檔案 |
| 還原 | 從備份還原伺服器 |

## 排程權限

| 權限名稱 | 說明 |
|----------|------|
| 建立 | 建立新排程任務 |
| 讀取 | 查看排程任務列表 |
| 更新 | 修改排程任務 |
| 刪除 | 刪除排程任務 |

## 資料庫權限

| 權限名稱 | 說明 |
|----------|------|
| 建立 | 建立新資料庫 |
| 讀取 | 查看資料庫列表 |
| 更新 | 修改資料庫密碼 |
| 刪除 | 刪除資料庫 |
| 檢視密碼 | 查看資料庫密碼 |

## 設定權限

| 權限名稱 | 說明 |
|----------|------|
| 重新命名 | 修改伺服器名稱 |
| 重新安裝 | 重新安裝伺服器 |

<Callout type="warn" title="權限提醒">
- 請謹慎授予「重新安裝」權限，此操作會清除所有伺服器資料
- 建議只給予子用戶必要的最小權限
</Callout>

# 管理現有子用戶

## 修改子用戶權限

1. 在使用者列表中找到要修改的子用戶
2. 點擊該用戶旁的編輯按鈕
3. 調整權限設定
4. 點擊「更新使用者」保存變更

![編輯子用戶](/images/panel/config/user/edit.png)

## 移除子用戶

1. 在使用者列表中找到要移除的子用戶
2. 點擊該用戶旁的刪除按鈕
3. 確認移除操作

![移除子用戶](/images/panel/config/user/delete.png)

<Callout type="info">移除子用戶後，該用戶將立即失去對此伺服器的所有存取權限。</Callout>

# 常見問題

### 子用戶找不到伺服器？
請確認您輸入的電子郵件地址正確，且對方已經註冊 CoffeeHost 帳號。

### 子用戶無法執行某些操作？
請檢查是否已授予對應的權限。子用戶只能執行您授權的操作。

### 可以新增多少位子用戶？
子用戶數量沒有限制，您可以根據需求新增任意數量的子用戶。
