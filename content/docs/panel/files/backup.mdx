# 創建備份
<Steps>
	<Step>
  
		### 點擊「備份」
    
    點擊之後會進入備份頁面，可以管理你的備份。

![圖片](/images/panel/files/backup/manage.png)

    </Step>
    <Step>
    ### 創建備份

    點擊「創建備份」開始創建

![圖片](/images/panel/files/backup/create.png)

    </Step>
    <Step>
	### 填入名稱
	為您的備份取一個名稱後按「開始備份」
	![圖片](/images/panel/files/backup/start.png)

</Step>
</Steps>

# 回復備份
<Steps>
	<Step>
  
		### 點擊「...」
    
    點擊之後可以根據您的需求進行下載、回復、鎖定、刪除等操作。


![圖片](/images/panel/files/backup/restore.png)

    </Step>
    <Step>
    ### 回復備份

    點擊「回復」後可決定是否清除原先資料，確定後按「Restore」。

![圖片](/images/panel/files/backup/confirm.png)

    </Step>
</Steps>
