<Steps>
	<Step>
  
		### 點擊「檔案管理」
    
    點擊之後會出現一個頁面，有很多檔案。

![圖片](/images/panel/files/edit/manage.png)

    </Step>
    <Step>
    ### 選擇檔案

    滑鼠左鍵點擊想編輯的檔案

![圖片](/images/panel/files/edit/choose.png)

    </Step>
    <Step>
	### 儲存檔案
	編輯完成後點擊 `儲存檔案`
	![圖片](/images/panel/files/edit/save.png)

</Step>
