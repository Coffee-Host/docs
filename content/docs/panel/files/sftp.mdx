---
title: SFTP
description: 使用 SFTP 上傳伺服器檔案。
---

控制面板的檔案管理僅供基礎使用，如需上傳大檔案或資料夾，就要使用 SFTP 來進行

import { Step, Steps } from "fumadocs-ui/components/steps";
import { Tab, Tabs } from "fumadocs-ui/components/tabs";
import { Callout } from "fumadocs-ui/components/callout";

<Steps>
	<Step>
	    ### 安裝客戶端
        如果您的電腦尚未安裝 SFTP 客戶端，可參考[此教學安裝 WinSCP](winscp)
    </Step>
    <Step>
        ### 取得連線資訊
        至[控制面板](https://panel.coffeehost.net)選擇您的伺服器，並進入「設定」分頁。
        ![圖片](/images/panel/files/sftp/settings.png)
    </Step>
    <Step>
        ### 連線至 SFTP
        若您的電腦已安裝 SFTP 客戶端，點擊「啟動 SFTP」即會自動開啟。
        ![圖片](/images/panel/files/sftp/sftp.png)
    </Step>
    <Step>
        ### 允許開啟
        您可勾選該核取方塊，這樣您下次就不用再允許一次，點擊「開啟 WinSCP」以繼續。
        ![圖片](/images/panel/files/sftp/confirm.png)
    </Step>
    <Step>
        ### 接受視窗
        第一次連線主機會出現此訊息，按「接受」以繼續連線。
        ![圖片](/images/panel/files/sftp/accept.png)
    </Step>
	<Step>
        ### 輸入密碼
        SFTP 密碼即為您登入控制面板使用的密碼。
        ![圖片](/images/panel/files/sftp/password.png)
	</Step>
	<Step>
        ### 開始使用
        成功連線後即可照一般管理檔案的邏輯上傳或下載檔案了
        ![圖片](/images/panel/files/sftp/done.png)
	</Step>
</Steps>
