## FileZilla

1. 下載 [FileZilla](https://filezilla-project.org/download.php?type=client)

2. 完成安裝並開啟 FileZilla

3. 將資訊填入上方欄位

4. 點擊 `快速連線` 或按下 Enter

## WinSCP

1. 下載 [WinSCP](https://winscp.net/eng/download.php)

2. 完成安裝並開啟 WinSCP

3. 將資訊填入欄位

4. 點擊 `登入` 或按下 Enter

## SFTP 資訊
主機資訊 和 使用者名稱 請至面板點擊 `設定` 查看
![圖片](/images/panel/files/sftp/position.png)

密碼和面板密碼相同
