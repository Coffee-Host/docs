---
title: WinSCP
description: 安裝 WinSCP 以使用 SFTP。
---

WinSCP 是在 Windows 中使用 SSH 的開放原始碼的圖形化 SFTP 用戶端。

安裝 WinSCP 後您將可以很輕鬆得透過 SFTP 管理伺服器上的檔案。

import { Step, Steps } from "fumadocs-ui/components/steps";
import { Tab, Tabs } from "fumadocs-ui/components/tabs";
import { Callout } from "fumadocs-ui/components/callout";

<Steps>
	<Step>
	    ### 下載安裝檔
        進入「[官方網站](https://winscp.net/eng/download.php)」並點擊「Download」下載安裝檔。
        ![圖片](/images/panel/files/winscp/download.png)
    </Step>
    <Step>
        ### 執行安裝檔
        下載完成後，開啟安裝檔並執行。
        ![圖片](/images/panel/files/winscp/open.png)
    </Step>
    <Step>
        ### 安裝類型
        若您的電腦沒有多使用者，點選「僅為我安裝」即可。
        ![圖片](/images/panel/files/winscp/uac.png)
    </Step>
    <Step>
        ### 授權合約
        點擊「接受」按鈕以繼續。
        ![圖片](/images/panel/files/winscp/contract.png)
    </Step>
    <Step>
        ### 操作介面
        保持建議的「典型安裝」即可，點擊「下一步」。
        ![圖片](/images/panel/files/winscp/type.png)
    </Step>
	<Step>
        ### 操作介面
        如果沒有使用過「雙窗」推薦選擇「檔案總管」操作較直覺，點擊「下一步」。
        ![圖片](/images/panel/files/winscp/view.png)
	</Step>
	<Step>
        ### 準備安裝
        點擊「安裝」按鈕以開始。
        ![圖片](/images/panel/files/winscp/confirm.png)
	</Step>
	<Step>
        ### 安裝完成
        點擊「完成」按鈕以關閉。
        ![圖片](/images/panel/files/winscp/complete.png)
	</Step>
</Steps>
