import { Callout } from "fumadocs-ui/components/callout";
import { Step, Steps } from "fumadocs-ui/components/steps";

<Callout title="警告" type="warn">

1. 超過 100MB 的檔案或者資料夾請使用 [SFTP](sftp)。
2. 無法透過此頁面上傳資料夾！

</Callout>

<Steps>
	<Step>
  
		### 點擊「檔案管理」
    
    點擊之後會出現一個頁面，有很多檔案。

![圖片](/images/panel/manage/upload/manage.png)

    </Step>
    <Step>
    ### 上傳檔案

    點擊「上傳」或將檔案直接拖曳到頁面上

![圖片](/images/panel/manage/upload/upload.png)

    </Step>

</Steps>
