## 步驟

1. 在面板點擊 ``

2. 點擊 `創建排程`

3. 設定排程，可以開啟 **顯示快速參考指令表** 來參考

4. 點擊 `管理排程`

5. 點擊 `新工作`

6. 設定完成後點擊 `創建任務`

## 範例

### 每日午夜自動重啟

    - 排程設定為 0 0 * * *
    - 任務動作選擇 `傳送電源操作`
    - 負載選擇 `重啟伺服器`
