---
title: 啟動設定
description: 設定伺服器啟動選項。
---

import { Callout } from "fumadocs-ui/components/callout";

在面板點擊 `啟動設定`

可以調整的參數如下:

#### MINECRAFT 版本

    - 可以更改您伺服器的 Minecraft 版本

#### 伺服器 JAR 檔

    - 可以更改您伺服器的 JAR 啟動檔案

#### Docker Image (映像檔)

    - 可以更改您伺服器的 Java 版本

#### BUILD NUMBER / Forge 版本

    - 可以更改您伺服器的核心版本(無須特別指定版本則可以留空)

#### BUILD TYPE(Forge)

    - 選擇您要下載 **推薦版** 或是 **最新版** 的 Forge 核心

設定完成後在面板點擊 `設定`，並選擇 `重新安裝伺服器`


<Callout title="警告" type="warn">
	請注意不同 Minecraft 版本對應的 Java 版本
</Callout>

<Callout title="警告" type="warn">
	如果要更改啟動指令請至 [Discord 伺服器](https://dc.coffeehost.net)
	開啟客服單。
</Callout>
