import { Callout } from "fumadocs-ui/components/callout";

<Callout title="仍在努力撰寫！" />

1. 在面板點擊 `啟動設定`

可以調整的參數如下:

### MINECRAFT 版本
    - 可以更改您伺服器的 Minecraft 版本

### 伺服器 JAR 檔
    - 可以更改您伺服器的 JAR 啟動檔案

### Docker Image (映像檔)
    - 可以更改您伺服器的 Java 版本

### BUILD NUMBER / Forge 版本
    - 可以更改您伺服器的核心版本

### BUILD TYPE(Forge)
    - 選擇您要下載 **推薦版** 或是 **最新版** 的 Forge 核心

設定完成後在面板點擊 `設定`，並選擇 `重新安裝伺服器`
