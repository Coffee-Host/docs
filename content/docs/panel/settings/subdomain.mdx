---
title: 子網域
description: 創建免費子網域。
---

import { Callout } from "fumadocs-ui/components/callout";
import { Step, Steps } from "fumadocs-ui/components/steps";

<Callout title="警告" type="warn">

子網域僅限 Java 版使用。

</Callout>

![圖片](/images/panel/settings/subdomain/create.png)

<Steps>
	<Step>
  
		### 點擊「子網域管理」
    
    點擊之後會出現一個頁面，會顯示創建和管理子網域的地方。

    </Step>
    <Step>

    ### 輸入子網域名稱

    輸入您想要的子網域，並於右側選擇想要的網域。

    </Step>
    <Step>

    ### 點擊「創建」

    創建後，您就可以使用這個網域連線，不需要加上連接埠。

    </Step>

</Steps>
