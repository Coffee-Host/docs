# 如何架設 Geyser

<Steps>
	<Step>
  
	### 下載 Geyser 插件

    到 [**Geyser 下載介面**](https://geysermc.org/download) 下載 Geyser，選擇您要的版本並安裝（以 Spigot/Paper 示範）。

    </Step>

    <Step>

    ### 將插件放入 `/home/container/plugins`

    放完插件後重啟伺服器後，會發現多了一個 `Geyser-Spigot` 資料夾。

    </Step>

    <Step>

    ### 打開 Geyser-Spigot 資料夾裡面的 config.yml

    將第 23 行的 `clone-remote-port` 從 `false` 調整為 `true`

    ![圖片](/images/practice/geyser/config.png)

    </Step>

    <Step>

    ### 儲存並套用

    儲存檔案後，重新啟動伺服器即可套用設定。

    </Step>

</Steps>

# 利用 Floodgate 使未購買電腦版的玩家直接加入伺服器

<Steps>
	<Step>
  
	### 下載 Floodgate 插件

    到 [**Floodgate 下載介面**](https://geysermc.org/download) 下載 Geyser，選擇您要的版本並安裝（以 Spigot/Paper 示範）。

    </Step>

    <Step>

    ### 將插件放入 `/home/container/plugins`

    放完插件後重啟伺服器即可。
    </Step>

</Steps>
