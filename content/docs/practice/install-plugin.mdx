# 什麼是插件？

插件（Plugin）是用來擴展 Minecraft 伺服器功能的程式。透過安裝插件，您可以為伺服器添加各種功能，例如：

- 權限管理（LuckPerms）
- 經濟系統（Vault、EssentialsX）
- 領地保護（WorldGuard、GriefPrevention）
- 小遊戲（Bedwars、Skywars）
- 登入系統（AuthMe）

# 在哪裡下載插件？

以下是一些常見且安全的插件下載平台：

| 平台 | 網址 | 說明 |
|------|------|------|
| SpigotMC | https://www.spigotmc.org/resources/ | 最大的插件平台 |
| Modrinth | https://modrinth.com/plugins | 新興平台，介面簡潔 |
| Hangar | https://hangar.papermc.io/ | Paper 官方平台 |
| Bukkit | https://dev.bukkit.org/ | 老牌平台 |

<Callout type="warn" title="安全提醒">請勿從不明來源下載插件，以避免伺服器被植入惡意程式。</Callout>

# 安裝插件教學

<Steps>
    <Step>

    ### 下載插件檔案

    1. 前往插件下載平台（以 SpigotMC 為例）
    2. 搜尋您想要的插件名稱
    3. 確認插件支援您的伺服器版本
    4. 點擊「Download」下載 `.jar` 檔案

    ![下載插件](/images/practice/install-plugin/download.png)

    <Callout type="info">請確認插件版本與您的伺服器版本相容。例如 1.20.4 的伺服器應下載支援 1.20.4 的插件版本。</Callout>

    </Step>

    <Step>

    ### 上傳插件到伺服器

    1. 前往面板的「檔案管理」頁面
    2. 進入 `plugins` 資料夾
    3. 點擊「上傳」按鈕
    4. 選擇剛才下載的 `.jar` 檔案並上傳

    ![上傳插件](/images/practice/install-plugin/upload.png)

    <Callout type="info">您也可以使用 SFTP 上傳，詳見 [SFTP 教學](/docs/panel/common/sftp)。</Callout>

    </Step>

    <Step>

    ### 重新啟動伺服器

    1. 回到「控制台」頁面
    2. 點擊「重新啟動」按鈕
    3. 等待伺服器完全啟動

    ![重新啟動](/images/practice/install-plugin/restart.png)

    </Step>

    <Step>

    ### 確認插件是否載入成功

    伺服器啟動後，在控制台輸入指令 `plugins` 或 `pl`，查看已載入的插件列表。

    - 綠色的插件名稱表示載入成功
    - 紅色的插件名稱表示載入失敗

    ![檢查插件](/images/practice/install-plugin/check.png)

    </Step>

</Steps>

# 插件載入失敗怎麼辦？

如果插件顯示為紅色或控制台出現錯誤，請檢查以下幾點：

1. **版本不相容**：確認插件支援您的伺服器版本
2. **缺少依賴插件**：部分插件需要其他插件才能運作（例如需要 Vault）
3. **檔案損壞**：重新下載插件檔案
4. **Java 版本問題**：部分新插件需要 Java 17 或以上版本

<Callout type="info">如果無法自行解決，可以使用 [ChatGPT 分析錯誤](/docs/troubleshooting/chatgpt) 來找出問題原因。</Callout>
