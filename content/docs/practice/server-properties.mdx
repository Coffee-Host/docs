<Callout title="什麼是 server.properties？">
`server.properties` 是 Minecraft 伺服器的核心設定檔，控制伺服器的基本行為和遊戲規則。
</Callout>

# 修改設定檔

<Steps>
    <Step>

    ### 關閉伺服器

    在修改 `server.properties` 之前，建議先關閉伺服器，以確保設定正確儲存。

    </Step>

    <Step>

    ### 開啟設定檔

    1. 前往面板的「檔案管理」頁面
    2. 在根目錄找到 `server.properties` 檔案
    3. 點擊檔案開啟編輯器

    ![開啟設定檔](/images/practice/server-properties/open.png)

    </Step>

    <Step>

    ### 修改設定

    根據需求修改相關設定（詳見下方設定說明）。

    ![修改設定](/images/practice/server-properties/edit.png)

    </Step>

    <Step>

    ### 儲存並重啟

    儲存設定檔後，重新啟動伺服器以套用變更。

    </Step>

</Steps>

# 常用設定說明

## 伺服器基礎設定

| 設定名稱 | 預設值 | 說明 |
|----------|--------|------|
| `motd` | A Minecraft Server | 伺服器列表顯示的描述文字 |
| `max-players` | 20 | 最大玩家人數 |
| `online-mode` | true | 正版驗證（false 允許盜版） |
| `white-list` | false | 白名單模式 |
| `enforce-whitelist` | false | 強制執行白名單 |

### MOTD 設定範例

```properties
# 支援顏色代碼和格式代碼
motd=\u00A7b歡迎來到我的伺服器！\u00A7r\n\u00A7e正在運行 1.20.4
```

<Callout type="info">
`\u00A7` 是顏色代碼的 Unicode 表示，常用顏色：
- `\u00A7a` 綠色、`\u00A7b` 青色、`\u00A7c` 紅色
- `\u00A7e` 黃色、`\u00A7f` 白色、`\u00A7l` 粗體
</Callout>

## 遊戲規則設定

| 設定名稱 | 預設值 | 說明 |
|----------|--------|------|
| `gamemode` | survival | 預設遊戲模式（survival/creative/adventure/spectator） |
| `difficulty` | easy | 遊戲難度（peaceful/easy/normal/hard） |
| `hardcore` | false | 極限模式（死亡後被 Ban） |
| `pvp` | true | 允許 PvP |
| `allow-flight` | false | 允許飛行（防止飛行踢出） |
| `spawn-protection` | 16 | 出生點保護範圍（格數） |

### 遊戲模式設定

```properties
# 生存模式
gamemode=survival
force-gamemode=false

# 創造模式伺服器
gamemode=creative
force-gamemode=true
```

## 世界設定

| 設定名稱 | 預設值 | 說明 |
|----------|--------|------|
| `level-name` | world | 世界資料夾名稱 |
| `level-seed` | （空白） | 世界種子碼 |
| `level-type` | minecraft\:normal | 世界類型 |
| `generate-structures` | true | 生成結構（村莊、神殿等） |
| `spawn-monsters` | true | 生成怪物 |
| `spawn-animals` | true | 生成動物 |
| `spawn-npcs` | true | 生成村民 |
| `allow-nether` | true | 允許地獄 |

### 世界類型選項

```properties
# 一般世界
level-type=minecraft\:normal

# 超平坦世界
level-type=minecraft\:flat

# 放大化世界
level-type=minecraft\:amplified

# 大型生態域
level-type=minecraft\:large_biomes
```

## 效能設定

| 設定名稱 | 預設值 | 說明 |
|----------|--------|------|
| `view-distance` | 10 | 伺服器視野距離（區塊） |
| `simulation-distance` | 10 | 實體模擬距離（區塊） |
| `max-tick-time` | 60000 | 最大 Tick 時間（毫秒） |
| `network-compression-threshold` | 256 | 網路壓縮閾值 |

### 效能優化建議

```properties
# 小型伺服器（1-10 人）
view-distance=10
simulation-distance=10

# 中型伺服器（10-30 人）
view-distance=8
simulation-distance=8

# 大型伺服器（30+ 人）
view-distance=6
simulation-distance=6
```

<Callout type="warn">降低視野距離可以顯著提升伺服器效能，但會影響玩家體驗。請根據實際情況調整。</Callout>

## 安全性設定

| 設定名稱 | 預設值 | 說明 |
|----------|--------|------|
| `enable-command-block` | false | 啟用指令方塊 |
| `op-permission-level` | 4 | OP 權限等級（1-4） |
| `function-permission-level` | 2 | 函數權限等級 |
| `enable-rcon` | false | 啟用遠端控制台 |

### OP 權限等級說明

| 等級 | 權限 |
|------|------|
| 1 | 可以繞過出生點保護 |
| 2 | 可以使用單人作弊指令和指令方塊 |
| 3 | 可以管理玩家（kick、ban、op） |
| 4 | 可以使用所有指令（stop、save） |

# 完整設定檔範例

```properties
#Minecraft server properties
# 基礎設定
motd=\u00A7b我的 Minecraft 伺服器\u00A7r\n\u00A7e歡迎加入！
max-players=50
online-mode=true
white-list=false

# 遊戲設定
gamemode=survival
difficulty=normal
pvp=true
hardcore=false
allow-flight=false

# 世界設定
level-name=world
level-seed=
generate-structures=true
spawn-monsters=true
spawn-animals=true

# 效能設定
view-distance=10
simulation-distance=10

# 安全設定
enable-command-block=false
spawn-protection=16
op-permission-level=4
```

<Callout type="info">修改設定後需要重新啟動伺服器才會生效。部分設定（如 `level-seed`）只在生成新世界時生效。</Callout>
