# Cloudflare SRV 紀錄設定教學

在開始之前，請至 Cloudflare 添加您的網域，並確認網域狀態為「使用中」。

<Steps>
	<Step>
  
	### 前往 Cloudflare DNS 管理介面

    進入 [**Cloudflare DNS 管理介面**](https://dash.cloudflare.com/) 後，選擇您的網域，並進入「DNS」頁面。

    </Step>

    <Step>

    ### 新增 SRV 紀錄

    請參考圖片內容填寫，以 `test.coffeehub.club` 綁定 `tsa.cfh.tw:11102` 為例。

    ![圖片](/images/practice/subdomain/config.png)

    </Step>

    <Step>

    ### 等待 DNS 更新

    完成後，請靜候 10 分鐘 ~ 24 小時，使 DNS 完全更新，即可使用您的網域直接連線。

    </Step>

</Steps>
