如您是首次購買，您將會在服務開通後收到控制面板帳號創建通知。

點擊中間 「開始設定您的帳號」，進入控制面板網頁。
![圖片](/images/quick-start/start-server/email.png)

輸入您帳號的新密碼後，點擊重設密碼即可開始使用。
![圖片](/images/quick-start/start-server/reset.png)

<Callout title="除了可以使用帳號密碼登入外，還可以透過 Google 或 Discord 等社群帳號登入。" type="info">只要社群帳號 Email 地址與控制面板帳號 Email 相同，即可快速登入！</Callout>

登入後即可看到伺服器列表，左下角可管理您的帳號。
![圖片](/images/quick-start/start-server/panel.png)

切換至「控制台」分頁。
![圖片](/images/quick-start/start-server/dashboard.png)

點擊「啟動」按鈕。
![圖片](/images/quick-start/start-server/console.png)

按「I Accept」以同意 Eula。
![圖片](/images/quick-start/start-server/eula.png)

看到「Done! For help, type "help"」即啟動成功。
![圖片](/images/quick-start/start-server/done.png)

點擊上方紅框複製連線位址。
![圖片](/images/quick-start/start-server/copy.png)

至遊戲貼上並連線。
![圖片](/images/quick-start/start-server/connect.png)
