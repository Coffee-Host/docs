title: 取消產品或服務

---

import { Callout } from "fumadocs-ui/components/callout";

<Callout title="申請取消表示您知道並同意取消服務免責聲明。" type="warn" />

如你尚有效的服務但下個月並不想續約或用不到， 可以提前取消訂閱，在到期前就不會再寄送催繳通知了！

import { Step, Steps } from "fumadocs-ui/components/steps";

# 申請取消
<Steps>
	<Step>
	    登入客戶中心。
        ![圖片](/images/store/service/cancel-service/login.png)
    </Step>
	<Step>
	    在首頁「您目前可用的產品與服務」區域選擇要取消的方案。
        ![圖片](/images/store/service/cancel-service/clientarea.png)
    </Step>
	<Step>
	    在左側動作找到「要求撤銷」並進入。
        ![圖片](/images/store/service/cancel-service/cancel.png)
    </Step>
	<Step>
	    輸入取消原因，也歡迎提供意見給我們！
        ![圖片](/images/store/service/cancel-service/reason.png)
    </Step>
	<Step>
	    最後點擊「要求撤銷」送出即可。
        ![圖片](/images/store/service/cancel-service/confirm.png)
    </Step>
	<Step>
	    成功送出申請。
        ![圖片](/images/store/service/cancel-service/success.png)
    </Step>
	<Step>
	    如果你突然改變心意，或是不小心送出錯誤申請，請立即聯絡我們！
    </Step>
</Steps>
