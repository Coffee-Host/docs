---
title: 升級服務方案
description: 升級或降級您的伺服器方案。
---

import { Callout } from "fumadocs-ui/components/callout";

<Callout title="如果您是想要更換方案系列，請至 Discord 聯絡客服。（像是 TW-L 升級到 TW-X 系列）" />

import { Step, Steps } from "fumadocs-ui/components/steps";

<Steps>
	<Step>
	    登入客戶中心。
        ![圖片](/images/store/service/upgrade-plan/login.png)
    </Step>
	<Step>
	    在首頁「您目前可用的產品與服務」區域選擇要升級的方案。
        ![圖片](/images/store/service/upgrade-plan/clientarea.png)
    </Step>
	<Step>
	    在左側的動作找到「升級/降級」並進入。
        ![圖片](/images/store/service/upgrade-plan/service.png)
    </Step>
	<Step>
	    進入頁面後，您就可以開始選擇要升級或降級的方案，升級的價格則會在下一頁計算。
        ![圖片](/images/store/service/upgrade-plan/choose.png)
    </Step>
	<Step>
	    確認付款金額與付款方式。如果您有優惠代碼，則可以在欄位內填入。最後點擊「結帳」。
        ![圖片](/images/store/service/upgrade-plan/checkout.png)
    </Step>
	<Step>
	    如果您有剩餘的帳號餘額，可以使用並立即完成訂單；如果沒有則系統將會導向到帳單頁面付款。
    </Step>
	<Step>
	    付款完成後將收到 Email 通知！（目前遊戲主機為自動完成升級）
    </Step>
</Steps>
