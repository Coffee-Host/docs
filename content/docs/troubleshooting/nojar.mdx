import { Callout } from "fumadocs-ui/components/callout";

當伺服器啟動時出現
`Error: Unable to access jarfile server.jar` 錯誤
![nojar](/images/troubleshooting/nojar/nojar.png)

到左側 **啟動設定**
![啟動設定](/images/troubleshooting/nojar/panel.png)

確認伺服器版本和核心版本填寫正確，可以參閱 [啟動設定](/docs/panel/config/startup)
![啟動設定](/images/troubleshooting/nojar/startup.png)

Forge 版本和 server.jar 正常情況下留空即可，只需更改 Minecraft 版本和 Java 版本

更改後到左側設定點擊 **重新安裝伺服器** 即可開啟
![啟動設定](/images/troubleshooting/nojar/reinstall.png)
